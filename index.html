    An open-source AutoML Library in PyTorch¶

    Installation¶

    Recommended¶

    pip install -U gradsflow

    From source¶

    pip install git+https://github.com/gradsflow/gradsflow@main

    What is GradsFlow?¶

    Attention

    GradsFlow is changing fast and is not stable yet.

    GradsFlow is based on Ray and PyTorch Lightning ‚ö°Ô∏è (support for other torch frameworks will be added soon). It leverages PyTorch Lightning Flash so that you don't have to write any PyTorch or Optuna code for model building or hyperparameter tuning üöÄ

    Although you might want to train a custom model and search hyperparameters, You can easily integrate any PyTorch/Lightning Flash Model with Gradsflow AutoModel ‚ú®

    • gradsflow.core: Core defines the building blocks of AutoML tasks.

    • gradsflow.autotasks: AutoTasks defines different ML/DL tasks which is provided by gradsflow AutoML API.
